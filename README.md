# Diabetes Risk Intelligence & Patient Analytics Dashboard (Power BI)

## Project Overview

Interactive Power BI healthcare dashboard analyzing 768 patients to identify diabetes risk factors, segment by clinical parameters, and enable early intervention.

## Tools Used

- Power BI Desktop
- DAX
- Power Query
- Data Modeling
- Healthcare Analytics

## Dataset

**Source:** Pima Indians Diabetes Database  
**Records:** 768 patients  
**Attributes:** Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age, Outcome

## Key Findings

| Finding | Value |
|---------|-------|
| Diabetes Prevalence | 34.9% |
| Top Risk Factor | Glucose (0.47 correlation) |
| Second Risk Factor | BMI (0.29 correlation) |
| Third Risk Factor | Age (0.24 correlation) |
| Average BMI | 31.99 |
| Average Glucose | 120.89 |

## Critical Insight

Patients with **7 or more pregnancies** have **56-64% diabetes prevalence**, compared to only 18-37% for lower pregnancy groups.

## Dashboard Features

- Risk Factor Analysis (correlation matrix)
- Patient Segmentation by glucose range, BMI category, age group
- Pregnancy Impact Analysis
- Dynamic Filtering (age group, BMI, outcome)
- Age vs. Glucose Trend Analysis

## Dashboard Pages

1. Diabetes Risk Intelligence & Overview
2. Risk Factor Analysis
3. Patient Segmentation
4. Prediction & Insights
5. Pregnancies Analysis

## How to View

1. Download the `.pbix` file from this repository
